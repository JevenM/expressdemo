## 项目来源
[全栈之巅UP主](https://www.bilibili.com/video/BV1Nb411j7AC?from=search&seid=13654016296741048646)

主要是一个使用
`express+nodejs+nodemon+mongoose+bcrypt+rest-client`
实现利用`token`来做用户认证的小`demo`

## 实现了接口
1、登录

2、注册

3、查看所有用户

4、某个用户资料

## 数据库
MongoDB


## 启动

node server.js

或者

nodemon server.js


## 打开

http://localhost:3001
